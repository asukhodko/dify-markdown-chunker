# Testing Guide

Guide for testing the Dify Markdown Chunker.

## Overview

The project includes comprehensive testing with 1366+ tests.

## Test Categories

1. **Unit Tests**: Component-level testing
2. **Integration Tests**: End-to-end workflows
3. **Property Tests**: Universal properties with Hypothesis

## Running Tests

```bash
# All tests
make test

# With verbose output
make test-verbose

# With coverage
make test-coverage

# Quick tests only
make test-quick
```

For complete developer documentation, see [Developer Guide](developer-guide.md).
