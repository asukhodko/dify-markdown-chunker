# Performance Guide

Performance characteristics and optimization tips.

## Benchmarks

| Document Size | Processing Time | Throughput | Chunks |
|--------------|----------------|------------|--------|
| 1 KB         | ~800 ms        | 1.3 KB/s   | 6      |
| 10 KB        | ~150 ms        | 66 KB/s    | 44     |
| 50 KB        | ~1.9 s         | 27 KB/s    | 215    |
| 100 KB       | ~7 s           | 14 KB/s    | 429    |

## Optimization Tips

- Use appropriate configuration profiles
- Enable caching for repeated processing
- Choose optimal chunk sizes for your use case

For complete developer documentation, see [Developer Guide](developer-guide.md).
