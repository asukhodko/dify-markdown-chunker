## ğŸ“– Documentation

### API Reference

#### Constructor

```python
new Ultracli(options)
```

**Options:**

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `apiKey` | string | required | Your API key |
| `timeout` | number | 30000 | Request timeout in ms |
| `retries` | number | 3 | Number of retry attempts |
| `debug` | boolean | false | Enable debug mode |

#### Methods

##### `doSomething()`

Performs the main operation.

```python
const result = await client.doSomething(params);
```

**Parameters:**

- `params` (Object): Configuration parameters

**Returns:** Promise<Result>

### Examples

#### Example 1: Basic Usage

```python
const client = new Ultracli({
    apiKey: process.env.API_KEY
});

const data = await client.getData();
console.log(data);
```

#### Example 2: Advanced Configuration

```python
const client = new Ultracli({
    apiKey: process.env.API_KEY,
    timeout: 60000,
    retries: 5,
    hooks: {
        beforeRequest: (config) => console.log('Request:', config),
        afterResponse: (response) => console.log('Response:', response)
    }
});
```
